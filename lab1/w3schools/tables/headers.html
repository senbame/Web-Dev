Table headers are defined with th elements. Each th element represents a table cell:
Firstname Lastname Age
Jill Smith 50
Eve Jackson 94
To use the first column as table headers, define the first cell in each row as a element:
Firstname Jill Eve
Lastname Smith Jackson
Age 94 50
Align Table Headers: To left-align the table headers, use the CSS text-align property: Example th { text-align: left; } Header for Multiple Columns:
Name Age
Jill Smith 50
Eve Jackson 94
You can add a caption that serves as a heading for the entire table:
Monthly savings
Month Savings
January $100
February $50
