Cell padding is the space between the cell edges and the cell content. th, td { padding: 15px; } Add padding to the other sides with the padding-bottom, padding-left, and padding-right properties: Example th, td { padding-top: 10px; padding-bottom: 20px; padding-left: 30px; padding-right: 40px; } Cell spacing is the space between each cell. To change the space between table cells, use the CSS border-spacing property on the table element: Example table { border-spacing: 30px; }
